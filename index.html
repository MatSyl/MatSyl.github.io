#5 - 📺 Arnaud Lemaire

  • Présentation : Arnaud Lemaire est un développeur et conférencier français, il parle de code, de legacy, d'agilité par l'extreme programming (XP) et de notre métier en général en synthétisant les idées développées par des grands noms du logiciel
  • Mots-clés : code / qualité / agilité
  • Blog : lilobase.me
  • Conférences :
    1. Entre industrialisation et artisanat, le métier de développeur
      • Public : programmeur et manager
      • Commentaires:
        • se concentrer sur les fondamentaux plutôt que sur les outils
        • "garder les options ouvertes"
        • mentionne le Toyota Production System 🚗 abordé dans XP
    2. Les microservices, c'est pas automatique
      • Public : programmeur
      • Commentaires: courte présentation (23 minutes), pleine de bon sens
    3. Et si on redémarrait l'agilité ?
      • Tout public 👐
      • Commentaires:
        • conférence autour des projets et de leurs acteurs
        • fait référence au Chaos Report qui présente l'état de l'art du développement logiciel (publié annuellement depuis 1994)
